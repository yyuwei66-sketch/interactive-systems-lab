## 🧠 System Design Approach

The project follows a **state-driven architecture**:

```text
Game States:
- Idle
- Playing
- Interaction
- Result
```

Core design principles:

* Separation of concerns (UI / Logic / State)
* Reusability of components
* Scalability across multiple applications
